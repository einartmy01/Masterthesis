To run the scripts:
1. Enter terminal
2. Navigate to folder with scripts
3. Execute in terminal: chmod +x "filename".
    3.1. This is a one time thing and does not need to be repeated. Only if you create a new sh file.
4. Execute in terminal: ./"filename".sh
    4.1. Always execute receiver first

Find IP:
1. Execute in terminal: ip -4 a
2. Find: "wlo1: inet 10.X.X.X"
3. Put that IP in sender code


Sender Local IP:
1. If sender don't have an Local enp0s31f6 adress
2. Follow these commands: 
    sudo ip addr flush dev enp0s31f6
    sudo ip addr add 192.168.2.50/24 dev enp0s31f6
    sudo ip addr flush dev eth0
    sudo ip addr add 192.168.0.50/24 dev eth0
    sudo ip addr flush dev eth1
    sudo ip addr add 192.168.1.50/24 dev eth1
